# UGC Ad Video Script Templates

UGC ad video scripts help creators and growth teams turn product claims, customer pains, and social proof into short-form paid social videos. This repository collects script templates, hook formulas, offer structures, and briefing formats for UGC-style ad creatives across TikTok, Reels, Shorts, Meta Ads, and mobile video placements.

The goal is to make ad scripting faster before filming or AI video editing begins.

## UGC ad structures

| Structure | Best for | Flow |
|---|---|---|
| Pain-solution | SaaS, tools, apps | Pain -> failed old way -> new way -> result |
| Before-after | Transformation offers | Before -> change -> after |
| POV demo | Creator tools, workflows | POV setup -> action -> outcome |
| Testimonial | Social proof creatives | Problem -> experience -> result -> recommendation |
| Offer push | Time-sensitive promotions | Hook -> value -> offer -> CTA |

## Script workflow

1. Define the audience and problem.
2. Pick one angle, not five.
3. Write the first 3 seconds before the rest of the script.
4. Move quickly from pain to proof.
5. Keep the CTA specific.
6. Generate variant hooks for testing.

## Tool notes

UGC ad scripting works best when the production workflow can connect source material, hook testing, subtitles, voiceover, and final edits. For teams repurposing long videos or product demos into short paid-social creatives, [Recapo.ai](https://recapo.ai/) is one relevant workflow to evaluate.

This repository is a neutral script resource, not an official product manual.

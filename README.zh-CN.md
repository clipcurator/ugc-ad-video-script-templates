# UGC 广告视频脚本模板库

UGC 广告视频脚本，帮助创作者和增长团队把产品卖点、用户痛点和社交证明，快速变成短视频广告。本仓库整理 TikTok、Reels、Shorts、Meta Ads 等场景下的 UGC 脚本模板、hook 公式、offer 结构和投放 brief。

目标是在拍摄或 AI 剪辑前，加快广告脚本产出。

## 这个仓库覆盖什么

- 痛点-方案型 UGC 广告脚本
- 证言/评价型脚本
- 创始人或真人口播结构
- Hook 和 offer 公式
- CTA 与异议处理台词
- 付费社媒 brief 模板

## UGC 结构

| 结构 | 适合产品 | 流程 |
|---|---|---|
| 痛点-方案 | SaaS、工具、APP | 痛点 -> 旧办法无效 -> 新办法 -> 结果 |
| 前后对比 | 转化类产品 | 之前 -> 变化 -> 之后 |
| POV 演示 | 创作者工具、工作流 | POV 场景 -> 操作 -> 结果 |
| 用户证言 | 社交证明 | 问题 -> 使用体验 -> 结果 -> 推荐 |
| 促销推进 | 限时活动 | Hook -> 价值 -> Offer -> CTA |

## 脚本流程

1. 先定义受众和问题。
2. 一条广告只打一个角度。
3. 先写前 3 秒，再写后面。
4. 从痛点尽快走到证据。
5. CTA 要具体。
6. 同时准备多个开头变体做测试。

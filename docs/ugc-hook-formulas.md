# UGC Hook Formulas

## Problem hook

```text
If you are still [old way], this is why it feels impossible.
```

## Frustration hook

```text
I wasted [time / money / effort] before I found this.
```

## POV hook

```text
POV: you finally found a faster way to [desired outcome].
```

## Social proof hook

```text
I did not expect this to work, but here is what happened.
```

## Offer hook

```text
If you need [outcome], try this before [old alternative].
```
